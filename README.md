## To start from codebase
```
yarn install
createdb 2ndDibsEcommerce
yarn seed
create a secrets.js file with:
  export const stripeAPITestKey = 'INSERT-YOUR-STRIPE-API-TEST-KEY-HERE';
yarn start-dev
```

## To test
```
yarn install
createdb 2ndDibsEcommerce-Test
yarn test
```
